About This Project

Welcome to the ArcGIS Map Viewer terminal interface for the final project. This final project displays a map of the NOAA CORRS. The NOAA CORRS is a network of continuously updating satellite refrence points called the global navigation satellite system or GNSS. These refrence points for meteorilogical surveys and data positioning. They can also be used for GPS positioning as well. The map displayed on this website is actually used to display working statuses for these sensors. The map will show all of the sensors in the U.S. and also shows their working status. This means they could be working, out of commision, or under maintenance .

Using an embedded ArcGIS map to create a viewable interactive experience for understnading NOAA CORRS refrence points as well as accompanying data..

Key Features

  • Interactive ArcGIS web map embedded with full functionality
  • Multiple data layers including activity levels for the reference points
  • Fullscreen mode for immersive map exploration
  • Direct query support with layer filtering
  • Easily accesible attribute data added in for further understanding
  • Cool design quality that promotes a fun experience
Note: This interface uses ArcGIS Web AppViewer to render geospatial data. All map interactions are handled through the embedded iframe with sandbox permissions enabled for security.

Map Layers

The system displays three primary data layers:

  • Active Sites - Current operational locations
  • Survey Areas - Designated survey zones
  • Basemaps & Tiles - Geographic reference data

Usage Instructions

Navigate the map using standard ArcGIS controls. Use the toolbar buttons to open the map in a new tab, toggle fullscreen mode, or reload the interface. Keyboard shortcut [F] activates fullscreen mode for rapid access.

Pro Tip: Press 'F' on your keyboard to quickly toggle fullscreen mode. Press Ctrl+R (Cmd+R on Mac) to reload the map.
